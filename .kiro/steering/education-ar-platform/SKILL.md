# 教育AR创作平台 - 产品技能（Skill）清单

本技能清单基于平台核心功能与用户需求制定，覆盖**创作能力、教学应用、平台运营**三大维度，用于指导用户使用及产品功能迭代。

---

## 一、核心创作技能

面向所有用户，聚焦零门槛完成AR内容制作的核心能力

### 1.1 资源上传与管理技能

| 技能 | 说明 | 等级 |
|------|------|------|
| 多类型素材上传 | 支持3D模型（.glb格式）、音频、图片、视频上传 | 入门 |
| 素材分类归档 | 按学科、场景、类型建立个人素材库 | 入门 |
| 素材引用能力 | 直接调用平台教育专属素材库资源 | 入门 |

### 1.2 场景编辑技能

| 技能 | 说明 | 等级 |
|------|------|------|
| 拖拽式场景搭建 | 添加、删除、移动虚拟物体 | 入门 |
| 物体属性调整 | 设置位置、大小、旋转角度、可见性 | 入门 |
| 实时预览 | 编辑过程中同步查看AR场景效果 | 入门 |

### 1.3 逻辑编排技能

| 技能 | 说明 | 等级 |
|------|------|------|
| 可视化逻辑拼图 | 通过拖拽节点搭建交互逻辑 | 进阶 |
| 触发器设置 | 配置"如果…就…"触发规则 | 进阶 |
| 多动作组合 | 实现播放音频、显示/隐藏物体、启动任务等多动作联动 | 进阶 |
| 延时任务设置 | 添加休眠时间，控制动作执行顺序 | 进阶 |

### 1.4 多模态交互设置技能

| 技能 | 说明 | 等级 |
|------|------|------|
| 手势交互配置 | 设置抓取、点击、滑动等手势触发对应动作 | 进阶 |
| 语音指令配置 | 添加语音命令，实现免手操作控制 | 精通 |
| 结构拆解/还原配置 | 针对教学模型设置分步展示逻辑 | 精通 |

### 1.5 作品导出与分享技能

| 技能 | 说明 | 等级 |

## 六、技能分级说明

### 入门级（基础操作）
适合首次接触AR创作的用户，掌握后可独立完成简单AR作品

**核心能力**：
- 素材上传与管理
- 基础场景搭建
- 作品分享与展示

**学习时长**：约2-4小时

### 进阶级（深度应用）
适合有一定基础的用户，掌握后可创作复杂交互的AR内容

**核心能力**：
- 逻辑编排与触发器
- 多模态交互设置
- 多人协作与设备互联

**学习时长**：约8-16小时

### 精通级（专业创作）
适合深度用户，掌握后可主导AR教学项目与资源建设

**核心能力**：
- 语音交互与结构拆解
- AI辅助创作
- 校本资源建设与赛事组织

**学习时长**：约40小时以上

## 九、WordPress 新闻服务集成

### 9.1 服务架构
平台使用 WordPress 作为后端新闻内容管理系统，通过直接调用 WordPress REST API 获取新闻数据。

### 9.2 API 接口说明

| 接口 | 方法 | 说明 |
|------|------|------|
| `/wp-json/wp/v2/posts` | GET | 获取新闻文章列表 |
| `/wp-json/wp/v2/categories` | GET | 获取新闻分类 |
| `/wp-json/wp/v2/posts/{id}` | GET | 获取单篇文章详情 |

### 9.3 新闻分类
- 官方新闻
- 教育AR行业动态
- 校园应用案例
- 平台更新公告

## 十、登录模态窗口

### 10.1 功能说明
平台采用模态窗口（Modal）形式实现用户登录，无需跳转外部页面，提升用户体验。

### 10.2 登录表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 用户名 | text | 是 | 支持用户名/手机号/邮箱 |
| 密码 | password | 是 | 密码输入，支持显示/隐藏切换 |

### 10.3 交互设计

**触发方式**:
- 点击导航栏"登录"按钮
- 点击Hero区"立即开始创作"按钮
- 点击CTA区"立即登录"按钮

**模态窗口内容**:
- 平台Logo
- 登录表单（用户名、密码）
- "记住我"复选框
- "忘记密码"链接
- 登录按钮
- 关闭按钮（右上角）

### 10.4 UI 规范
- 使用 Element Plus 的 `el-dialog` 组件
- 居中显示，背景遮罩
- 支持点击遮罩关闭
- 支持 ESC 键关闭
- 移动端全屏显示
